# BASLER_CAM(BASLER acA2500 - 14gm)

## Pylon Viewer (Setup the IP)

### 1. 設置 IP (GigE Vision)

- 打開乙太網路屬性
- 選擇 TCP/IPv4 並點擊屬性
- 選擇 "使用下列的 IP 地址"：
  - **IP 地址**：10.10.10.10
  - **子網路遮罩**：255.255.255.0
- 打開 Pylon IP Configurator
- 選擇 "Static IP"
  - **IP 地址**：10.10.10.11 (前三位要相同)
  - **子網路遮罩**：255.255.255.0
- 點擊 "保存"
